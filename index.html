Query

  connections(from: $from, to: $to, limit: $limit, offset: $offset) {

Nft

Field Name Description
address - String!
name - String!
sellerFeeBasisPoints - Int!
mintAddress - String!
primarySaleHappened - Boolean!
description - String!
image - String!

Arguments

width - Int

Image width possible values are:

  • 0 (Original size)
  • 100 (Tiny)
  • 400 (XSmall)
  • 600 (Small)
  • 800 (Medium)
  • 1400 (Large)

Any other value will return the original image size.

If no value is provided, it will return XSmall
creators - [NftCreator!]!
attributes - [NftAttribute!]!
owner - NftOwner
activities - [NftActivity!]!
listings - [ListingReceipt!]!
